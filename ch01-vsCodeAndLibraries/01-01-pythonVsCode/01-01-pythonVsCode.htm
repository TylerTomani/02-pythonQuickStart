Visual Studio Shorcuts

When Visual Studio Code is open and no window is showing, go to File → "New Text File", or just press,

⌘ + N

 To create a new empty file,
Press,

⌘ + N

again.

If the file is new hit ⌘ + S to save it and name it, or Press,

⌘ + ⇧(shift) + S

to save and rename an existing one.

If you want to change to the Desktop while in the finder window,
Press,

⌘ + ⇧(shift) + D

To Create a New folder while in the finder window,
Press,

⌘ + ⇧(shift) + N
